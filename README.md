# Zerodha Trading Platform Clone

A full-stack clone of , India's leading stock trading platform. This project simulates the core user experience of the Zerodha Kite dashboard, demonstrating a modern MERN stack application with a focus on a clean UI and a robust backend API.

---

## About The Project
This application was built to replicate the essential features of a modern trading dashboard. It serves as a practical showcase of full-stack development skills, including frontend state management with React, RESTful API design with Express, and data persistence using MongoDB. The project fetches and displays mock data for a user's portfolio, including their holdings and order history, in a clean, responsive interface.

## ✨ Features

- **Dashboard**: Clean and user-friendly layout for trading overview  
- **Orders Table**: View and manage mock or fetchable orders  
- **Holdings Page**: Track portfolio holdings  
- **Responsive Design**: Optimized for desktop and mobile  
- **Real-Time Data Fetching**: Simulated backend responses  
- **REST API**: Built with Express.js for backend endpoints  
- **MongoDB Storage**: Persist user data and mock trading information  
- **CORS Enabled**: Secure API access

## 💻 Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/zerodha-clone.git
cd zerodha-clone
```

## Install dependencies
```bash
npm install
```

## Start the backend server
```bash
npm run server
```

## Start the frontend
```bash
npm start
```
